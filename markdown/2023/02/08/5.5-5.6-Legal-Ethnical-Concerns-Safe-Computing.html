Big Idea 5.5 Legal and Ethnical Concerns

  • MIT License: Involves closed sources and the Github project can be private.
  • Open Source GPL License: People can do anything they want except distributing closed source versions. Github project is public, can people can be sued or forced to reveal their code if they use open source license.
  • Summary: Based on the discussions, it seems that MIT License is the best choice if you want your code to be private. You can also get credit with MIT License. There has been times where people have been sued or forced to reveal their code because they had a open source license.

Another step for authentication could be to use a finger print or face id. Symmetric encryption is where only one secret key is used to encrypt and decrypt info. Asymmetric cryptography uses one public key and one private key to encrypt and decrypt a message and protect it from unauthorized access.
